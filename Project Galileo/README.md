# Galileo - Control de Ventas

Proyecto de ejemplo: sistema de control de ventas llamado **Galileo**.

- Backend: Django + Django REST Framework
- Frontend: Plantilla HTML con Bootstrap para una interfaz visual y cómoda

Instrucciones rápidas:

1. Crear entorno virtual (Windows PowerShell):

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

2. Migrar y ejecutar servidor:

```powershell
python manage.py migrate
python manage.py runserver
```

3. Abrir en el navegador: http://127.0.0.1:8000/
